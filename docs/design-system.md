# HYBRID Home Assistant — Design System v1.0

## Princípios

- **Tecnologia acolhedora:** visual avançado sem parecer técnico ou complexo.
- **Clareza operacional:** estados, alertas e controles devem ser compreendidos rapidamente.
- **Local-first:** a interface informa claramente quando o sistema está local, remoto ou offline.
- **Consistência:** os mesmos padrões visuais são usados no dashboard, mobile e futuras aplicações.
- **Acessibilidade:** foco visível, contraste adequado e alvos interativos confortáveis.
