# HEF Glossary

- Home: A managed residence.
- Room: Physical area within a home.
- HomeEntity: Logical representation of an object or service in a room.
- HardwareBinding: Association between a HomeEntity and a physical device.
- Capability: A feature exposed by an entity.
- Driver: Adapter implementing capabilities for specific hardware.
- Fingerprint: Stable hardware identity used for matching.
- Digital Twin: Persistent logical representation of the home.
- Entity Registry: Catalog of all HomeEntity instances.
