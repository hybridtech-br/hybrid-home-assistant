# @hha/foundation

Contratos e tipos fundamentais independentes dos domínios de automação residencial.

## Incluído nesta versão

- Identificadores tipados.
- Validação básica de identificadores.
- `Result<T, E>`.
- `Clock`, `SystemClock` e `FixedClock`.
- Contrato imutável `HouseEvent<TPayload>`.

## Não incluído

Kernel, dispositivos, drivers, Digital Twin, persistência e inteligência.
